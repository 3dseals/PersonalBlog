# Product Overview

Personal technical blog for Lional, a developer based in Chengdu, China.

## Purpose
- Share knowledge on game development, backend systems, and development tools
- Document technical experiences with frameworks and architectures
- Publish tutorials and comparisons of development technologies

## Content Focus
- Game programming (Cocos2D-X, Unity3D, C++, Lua)
- Backend development (Golang, microservices, databases)
- DevOps and containerization
- AI-assisted programming tools
- Management practices (OKR)

## Target Audience
Chinese-speaking developers interested in game development, backend systems, and modern development practices.
